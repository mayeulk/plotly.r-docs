### Global Font Properties


```r
library(plotly)

t <- list(
  family = "sans serif",
  size = 14,
  color = 'blue')

fig <- plot_ly(x=c(1,2,3,4,5), y=c(1,2,3,2,1))
fig <- fig %>% layout(title="Font Styling",
         font=t)


fig
```
